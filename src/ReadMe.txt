For Using program:

1. You can re-make the dictionary file ( big.txt ) for your usage.
2. Make your own tweet files for sentiment analysis (Ex: Positive , Negative , Notr )
3. You can correct words in a hole file or you can correct one-by-one your incorrect words.

Console:

1. 'halt' kapatmak icin...     :  Use 'halt' for close the console
2. Kelimeleri Texte Aktar: 'getwcounts'  : Use 'getwcounts' for transfering all corrected words to a text file
3. Kelimeleri Excele Aktar: 'egetwcounts'  :  Use 'egetwcounts' for transfering all corrected words to a excel file
4. Arttribute Secim: 'getatt'  :  Use 'getatt' for making an arff file for WEKA
5. Tum Twitleri Duzelt: 'getfix' : Use 'getfix' for correcting all words of your main tweets file
6. Tek Bir Kelime: 'sfix' :  Use 'sfix' for correcting a single word.
